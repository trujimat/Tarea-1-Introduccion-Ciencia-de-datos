# Introducción a la Ciencia de Datos — Tarea 1

Este repositorio contiene la solución desarrollada para la Tarea 1 del curso *Introducción a la Ciencia de Datos*.

El archivo `Tarea1.ipynb`, ubicado en el directorio raíz del repositorio, incluye el código utilizado para el procesamiento de datos, generación de visualizaciones y obtención de los resultados solicitados en la tarea.

El archivo `Tarea1.pdf`, ubicado en el directorio raíz del repositorio, contiene las respuestas, el análisis de resultados y las visualizaciones gráficas y numéricas correspondientes a las distintas partes de la tarea.

## Requisitos

Se recomienda utilizar un entorno virtual de Python antes de instalar las dependencias del proyecto.

Las librerías necesarias se encuentran especificadas en el archivo `requirements.txt`.

## Instalación de dependencias

Ejecute el siguiente comando desde el directorio raíz del repositorio:

```bash
pip install -r requirements.txt

## Ejecución del notebook

Para reproducir los resultados obtenidos en la tarea, ejecute las celdas del notebook `Tarea1.ipynb` de manera secuencial utilizando el entorno de su preferencia, como Jupyter Notebook, Google Colab o Visual Studio Code.
